People Counter

Computer Vision to detect and count people by romualdjunior

Count the number of people using computer vision

This project uses YoloV5 to detect and count the number of people in an image or video. It is made using Python 3.8, Flask, and OpenCV. Take in mind this project is hosted on digitalocean It means sometimes the app may be slow to load, please be patient.
You have two options to use this project. The first is to upload an image and wait for the model to counter the number of people in it.
The last option is just an example of a yolo model counting people on a video camera .
